Documentation

Grid System

  • .container: Creates a fixed-width container.
  • .row: Creates a flex container for columns.
  • .col: Equal-width columns; use .col-sm-6 for 50% width on small screens and .col-md-4 for 33.33% on medium screens.

Buttons

  • .btn: Base button styling.
  • .btn-primary: Primary button color.
  • Add hover effects by customizing styles as needed.

Cards

  • .card: Creates a card component.
  • .card-body: For padding and content inside cards.

Forms

  • Style input fields with input[type="text"], input[type="email"], and input[type="password"].
  • Use select for dropdowns.

Utility Classes

  • .text-center: Centers text.
  • .mt-3: Adds top margin.
  • .mb-3: Adds bottom margin.
