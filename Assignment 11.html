CSS Positioning Techniques

Static Positioning (Default)

This box is statically positioned. This is the default positioning method where elements appear in the normal document flow.

Relative Positioning

This box is relatively positioned. It is shifted 20px down and 20px to the right from its original position without affecting other elements.

Absolute Positioning

This box is absolutely positioned. It is positioned 50px from the top and 50px from the left of the nearest positioned ancestor (the container).

Fixed Positioning

This box is fixed. It remains in the same position relative to the viewport even when the page is scrolled.

Sticky Positioning

This box is sticky. It sticks to the top of the viewport when you scroll down and reaches the top of the screen.
